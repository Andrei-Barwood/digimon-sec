# Instalación - Ciphermon

## Requisitos
- Python 3.10+
- pip o poetry

## PyPI (cuando se publique)
```bash
pip install ciphermon
```

## Código fuente
```bash
git clone https://github.com/yourusername/digimon-sec-suite.git
cd digimon-sec-suite/digimons/ciphermon
python -m venv venv
source venv/bin/activate
pip install -e ".[dev]"
```
